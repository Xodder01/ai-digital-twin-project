AI Digital Twin – Productivity & Burnout Prediction System

AI Digital Twin is a full-stack web application that uses Machine Learning to simulate a digital representation of a user's work behavior. The system analyzes daily activity inputs and predicts two important insights:

• Productivity Score
• Burnout Risk Level

The goal of this project is to help users understand their work patterns, identify signs of burnout early, and make better decisions to improve performance and well-being.

This project combines Artificial Intelligence with modern web development technologies to demonstrate how data-driven insights can be used in real-world productivity monitoring systems.

Project Overview

A Digital Twin is a virtual representation of a real-world entity. In this project, the Digital Twin represents a user's work habits and performance patterns.

The system collects user inputs such as working hours, sleep duration, stress level, and tasks completed. These inputs are processed by trained Machine Learning models to generate predictions.

The web interface provides a clean and user-friendly dashboard where users can view predictions, track past results, and understand their productivity trends.

Key Features

• Predict productivity score using Machine Learning
• Estimate burnout risk level
• Interactive and modern user interface
• Dashboard with productivity overview
• History page to track previous predictions
• Login interface for future authentication integration
• Responsive design for different screen sizes
• Component-based frontend architecture
• REST API backend for model predictions

How It Works

User enters daily activity details in the prediction form.

Frontend sends the data to the Flask API in JSON format.

The backend processes the data using trained Machine Learning models.

The system returns predictions including productivity score and burnout risk.

Results are displayed visually in the user interface.
